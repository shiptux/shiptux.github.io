##安装NextCloud

  1. 更新软件包

    dnf update

  2. 安装nextcloud

    dnf install -y nextcloud-nginx.noarch nextcloud.noarch

  3. 安装数据库 mariadb

    dnf install -y mariadb mariadb-server

  4. 启动

    systemctl start nginx php-fpm mariadb

  5. 开机默认启动

    systemctl enable nginx php-fpm mariadb

##添加用户，设置权限

  1. 创建用户命令

    CREATE USER ‘fedora‘@’%’ IDENTIFIED BY ‘fedora’;

  2. 创建数据库

    create database fedora;

  3. 分配权限

    GRANT ALL ON fedora.* TO ‘fedora’ WITH GRANT OPTION;

  4. 外网登录

    GRANT ALL privileges ON . TO fedora@’%’ identified by ‘fedora’;

  5. 授予权限

    grant all privileges on . to fedora identified by ‘fedora’ with grant option;
